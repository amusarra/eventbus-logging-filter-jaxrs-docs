= Download della Documentazione
:description: Scarica la documentazione del progetto in formato PDF o HTML5

La documentazione di questo progetto è disponibile per il download in più formati.
I file vengono generati automaticamente ad ogni push sul branch `main` tramite la pipeline CI/CD e
pubblicati insieme al sito Antora su GitHub Pages.

[NOTE]
====
I link di download puntano ai file pubblicati su GitHub Pages.
Se visualizzi questa pagina in locale (build Antora locale), i file potrebbero non essere
ancora stati generati. Esegui il workflow CI per generarli oppure consultali direttamente
dalla scheda https://github.com/amusarra/eventbus-logging-filter-jaxrs-docs/actions[Actions,window=_blank]
del repository.
====

== Generazione manuale

È possibile generare i formati localmente tramite Maven:

.Genera il PDF
[source,bash]
----
mvn asciidoctor:process-asciidoc@asciidoc-to-pdf
# Output: target/generated-pdf/eventbus-logging-filter-jaxrs-docs-1.0.0.pdf
----

.Genera l'HTML5
[source,bash]
----
mvn asciidoctor:process-asciidoc@asciidoc-to-html
# Output: target/generated-html5/
----

.Genera entrambi i formati in un unico comando
[source,bash]
----
mvn generate-resources
----
