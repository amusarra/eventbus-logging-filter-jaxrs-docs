== Quick Start

Per eseguire il progetto in locale in modalità sviluppo:

[source,shell]
----
# Clona il repository
git clone {github-url}
cd {project-name}

# Avvia Quarkus in modalità dev (Dev Services avviano automaticamente MongoDB e AMQP)
./mvnw quarkus:dev
----
